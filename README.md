# Spring Boot REST API

This repository contains my practice work while learning **Spring Boot**. It focuses on building RESTful APIs and understanding backend development using Spring framework components.

---

## 📚 About

The purpose of this project is to learn how to develop backend services using Spring Boot, implement REST APIs, and handle client requests effectively.

---

## 🚀 Topics Covered

* Building REST APIs in Spring Boot
* Spring MVC Architecture
* CRUD API Development
* Request Mapping
* Request Validation
* Handling HTTP Methods (GET, POST, PUT, DELETE)

---

## 🛠️ Tech Used

* Java
* Spring Boot
* Spring MVC
* Maven / Gradle

---

## 🎯 Learning Outcome

Through this project, I learned:

* How REST APIs work
* Backend development fundamentals
* API request & response handling
* Validation of client requests
* Structuring Spring Boot applications
